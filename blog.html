Frequently Asked Questions

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page as a structured tree, where elements are nodes that can be manipulated using JavaScript. Developers use the DOM to dynamically update content, styles, and attributes without reloading the entire webpage.

2. How do you select an element from the DOM?

To select an element from the DOM, use methods like document.getElementById("id"), document.querySelector(".class"), or document.querySelectorAll("tag"). Example: const element = document.querySelector(".my-class"); This allows you to manipulate the selected element’s content, styles, or attributes dynamically.

3. What is event delegation?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements, you attach one to the parent. Example: parent.addEventListener("click", (event) => { if (event.target.matches(".child")) console.log("Child clicked"); });

4. How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element’s attributes and styles using the DOM, use setAttribute() and getAttribute() for attributes, and modify element.style for inline styles. For example: element.setAttribute("class", "new-class"); element.style.color = "red"; You can also use classList.add() for efficient class management.
